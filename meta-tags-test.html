‚úÖ Implementation Details

  • Dynamic Detection: The system detects the domain using window.location.hostname
  • Automatic Updates: Meta tags are set dynamically when the Vue app loads
  • SEO Friendly: Each domain has its own canonical URL and appropriate language tags
  • Social Sharing: Open Graph and Twitter Card tags ensure proper previews on social media

How it works:

  1. User visits either waiwai.diy or wai.sale
  2. Vue app detects the domain in src/i18n/index.js
  3. Appropriate locale is set (ru for waiwai.diy, en for wai.sale)
  4. App.vue uses @vueuse/head to dynamically update all meta tags
  5. Social media crawlers see the correct language meta tags

üîç Testing Instructions

To verify the implementation works correctly:

  1. Local Testing: The development server will default to Russian. To test English, you would need to modify the domain detection temporarily.
  2. Production Testing: Deploy to both domains and use social media debugging tools:
    • Facebook Sharing Debugger
    • Twitter Card Validator
    • LinkedIn Post Inspector
  3. Browser Testing: View page source after the Vue app loads to see the dynamically set meta tags
